# 下载AVA-Kinetics视频

🐱‍🏍`download.py`，下载训练集、测试集、验证集分别用test.csv；train.csv；val.csv，下载所有文件使用all.csv。

```python
python download.py  --input_csv <csvfile> --output_dir <your/output>
```
例如
```
python download.py  --input_csv test.csv --output_dir test/
```
